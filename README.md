# IMU Skill Development Portfolio


The purpose of this repository is to outline my development of technical skills with inertial measurement units. This is done primarily through the jupyter notebook, which demonstrates my progress from manual processing of a single joints data to the automated pipeline
at the backend of the notebook.

Along the way I reflected on issues or questions I had, and how they were corrected in the next step. 

I also have all files used stored in the assets folder.
